## August 2019 to March 2020

	- Project initiated by Billy Albina, then of Bard College, as Open Cataloging Rules
	- Core Committee develops the project’s vision, scope, principles, and draft element set

## April 2020-December 2020

	- Project on hiatus

## 2021

	- Project revived by Denise Soufi, UNC-Chapel Hill, and Faye Leibowitz, formerly of the University of Pittsburgh
	- Resources for developing rules compiled
	- Publication platforms researched

## 2022

	- Series rules developed
	- Website developed on GitHub with help from the RBMS RDA Editorial Group
	- Name changed to Open Rules for Cataloging (ORC)

## 2023

	- Series rules published to the website
  - ORC incorporated and received federal non-profit status
	- First Executive Board formed

## 2024

	- Title rules published to the website
  - First elected Executive Board
  - Core Committee reorganized to Rules and Tech Committees

## 2025

	- Edition rules published to the website
	- Description Convention Source Code “orc” assigned by the [MARC Standards Office](https://www.loc.gov/marc/relators/tn250925src.html){:target="_blank"}
  - Imprint rules developed 
  - Material Description draft started
  - ORC becomes an Open Library Foundation incubation partner
